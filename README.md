# Pulsate Website

Website for the Pulsate project. Build with [Remix](https://remix.run/).

This site lives at [pulsate.dev](https://pulsate.dev).

## Contributing

### Getting Started

The website is built with [Remix](https://remix.run/). To get started, clone the repository and run the following commands:

```sh
pnpm install
pnpm dev
```
